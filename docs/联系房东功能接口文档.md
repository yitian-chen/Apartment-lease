# 联系房东功能 - 前端开发指南

## 一、功能概述

在房间详情页添加"联系房东"按钮，点击后弹窗展示房东信息（手机号、平台账号），用户可选择打电话或在平台内发起聊天。

---

## 二、数据库变更

已在 `room_info` 表添加 `landlord_id` 字段，关联 `user_info.id`。

---

## 三、接口变更

### 3.1 管理端 - 获取房东列表

用于管理端在编辑房间时选择房东。

```
GET /admin/room/listLandlords
```

**响应**
```json
{
  "code": 200,
  "msg": "成功",
  "data": [
    {
      "id": 1,
      "phone": "13800138000",
      "nickname": "张三",
      "avatarUrl": "https://xxx.com/avatar.png"
    }
  ]
}
```

```
GET /app/room/getDetailById?id={roomId}
```

**响应新增字段**
```json
{
  "code": 200,

  "data": {
    "id": 1,

    "rent": 3000.00,
```

**注意**：`landlordInfo` 仅在以下条件满足时返回：
- 房间已发布 (`is_release == RELEASED`)
- 房东已设置 (`landlord_id != null`)

## 四、前端实现建议

### 4.1 移动端 - 房间详情页

1. 在房间详情页添加"联系房东"按钮
2. 调用 `/app/room/getDetailById` 获取 `landlordInfo`
3. 如果 `landlordInfo` 存在，显示按钮；否则隐藏

### 4.2 联系房东弹窗

点击按钮后展示弹窗，内容包括：

| 字段 | 说明 |
|------|------|
| `avatarUrl` | 房东头像 |
| `nickname` | 房东昵称 |
| `phone` | 房东手机号 |

**弹窗交互：**

| 按钮 | 行为 |
|------|------|
| 拨打电话 | 调用 `tel:13800138000` 打开拨号 |
| 平台聊天 | 传入 `landlordInfo.userId` 作为 `toId`，通过 WebSocket 发送消息 |

### 4.3 WebSocket 聊天消息格式

```javascript
// 发送消息
{
  "toId": 1,           // 房东用户ID (landlordInfo.userId)
  "message": "你好，我想咨询租房"  // 消息内容
}

// 接收消息
{
  "fromId": 1,         // 发送者用户ID
  "message": "你好，有什么可以帮你"  // 消息内容
}
```

## 五、字段对照表

### 移动端 LandlordInfoVo

| 字段 | 类型 | 说明 |
|------|------|------|
| `userId` | Long | 房东用户ID，用于发起聊天 |
| `phone` | String | 房东手机号 |
| `nickname` | String | 房东昵称 |
| `avatarUrl` | String | 房东头像URL |

---

## 六、注意事项

1. **房东可见性**：未发布的房间不会返回房东信息
2. **缓存**：房间详情有 Redis 缓存，更改房东后最多几分钟内生效
3. **账号状态**：只有状态正常的房东(`status=ENABLE`)才会返回
